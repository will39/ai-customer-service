\# 我的店铺智能客服

这是一个基于大模型的电商智能客服系统。



\## 功能特点

1\. \*\*🎨 可配置后台\*\*：左侧侧边栏可在线修改AI设定、新增/删除快捷按钮。

2\. \*\*💬 智能对话\*\*：支持上下文多轮对话，专业解答电商问题。

3\. \*\*🚀 快速部署\*\*：使用Streamlit构建，一键启动即可使用。



\## 运行方法

1\. 安装依赖：`pip install -r requirements.txt`

2\. 运行程序：`streamlit run chat\_app.py`



\# 配置说明

1\. 新建 .env 文件

2\. 文件内添加：

DASHSCOPE\_API\_KEY=你自己的API\_KEY
